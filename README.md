# Reactivities

Reactivities is a simple web application for creating and exploring events.  
Users can browse activities, join them, or create their own.

## Features

- View a list of events  
- Create and manage your own activities  
- Join and leave events  
- User registration and login  
- Basic profile management  

## Tech Stack

- Backend: ASP.NET Core Web API  
- Database: Entity Framework Core  
- Frontend: React, TypeScript

## Purpose

This project was built as part of a course to understand how a full-stack application works, including backend, frontend, and their interaction.
